# Turbo Rush - Death Road

A browser-based racing game built with plain HTML, CSS, and JavaScript.

This version has been split into separate files for easier editing and GitHub sharing, and the developer credit has been updated to include `Samay Chutani`.

## Files

- `racing_game_v5.html` - main game page
- `racing_game_v5.css` - all game styles
- `racing_game_v5.js` - game logic, animations, controls, and overlays

## Features

- Arcade-style racing gameplay
- Traffic and zombie game modes
- Vehicle selection screen
- HUD, pause menu, and win/lose overlays
- Touch controls for mobile play
- Theme toggle

## How to Run

1. Download or clone this repository.
2. Keep `racing_game_v5.html`, `racing_game_v5.css`, and `racing_game_v5.js` in the same folder.
3. Open `racing_game_v5.html` in any modern web browser.
